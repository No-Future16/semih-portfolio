## What is a Markdown file?

You are reading one right now. A Markdown file (`.md`) is a plain text file with simple formatting rules. You do not need any special software to open it — Notepad, TextEdit, or any text editor works. Many tools also render it nicely (GitHub, Notion, VS Code).

Think of it as a simple way to write structured notes. The `#` symbols create headings, `-` creates bullet points, and `**this**` makes text **bold**.

## Installing Google Antigravity

1. Go to [antigravity.google](https://antigravity.google/)
2. Download the version for your operating system (Mac, Windows, or Linux)
3. Open the installer and follow the steps — it works like installing any normal app
4. Sign in with your Google account
5. You are ready to go

Antigravity is **free** during its public preview. You get generous usage limits with Gemini (Google's AI model). The limits reset every 5 hours if you run out.

---

## What does Antigravity actually do?

Antigravity is an AI coding tool. Instead of writing code yourself, you describe what you want in plain language, and the AI builds it for you. It can:

- Generate HTML, CSS, and JavaScript files
- Show you a live preview of your site
- Make changes based on your follow-up instructions ("make the background darker", "add a section for my projects")
- Plan the work before doing it — you can review and approve the plan

You do not need to know how to code. You need to know how to describe what you want clearly.
