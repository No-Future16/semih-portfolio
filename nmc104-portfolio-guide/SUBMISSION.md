# NMC104 – How to Submit Your Portfolio

## Step 1: Check that your site works

Before submitting, open your site in a browser and check:

- [ ] The page loads without errors
- [ ] All three sections are present (About, Work, Contact)
- [ ] Images display correctly (if you added any)
- [ ] The site looks reasonable on a smaller window (resize your browser to test)

---

## Step 2: Find your files

In Antigravity, your project files are saved in the Workspace folder you created. To find them:

1. In Antigravity, go to your project
2. Look for the folder location shown at the top of the editor
3. Open that folder in your file manager (Finder on Mac, File Explorer on Windows)

Your folder should contain at minimum:
- `index.html` — this is your homepage
- A CSS file (e.g. `style.css` or `styles.css`)
- Possibly a JavaScript file (e.g. `script.js`)
- Any images you used

---

## Step 3: Create your ZIP file

1. Select your entire project folder
2. Right-click → **Compress** (Mac) or **Send to → Compressed folder** (Windows)
3. Name the ZIP file exactly like this:

```
NMC104_Portfolio_YourName.zip
```

Replace `YourName` with your actual name, no spaces. For example:
`NMC104_Portfolio_AyseKaya.zip`

---

## Step 4: Upload to STiX

1. Go to [stix.uskudar.edu.tr](https://stix.uskudar.edu.tr)
2. Find the **NMC104** course
3. Go to the **Portfolio** assignment tab
4. Upload your `.zip` file before the deadline

---

## Important notes

- Submit only once. If you need to resubmit, contact your instructor before the deadline.
- Do not submit just the `index.html` file. Submit the entire folder as a ZIP.
- Do not submit a link to an online site. Submit the files.
- If your ZIP is larger than expected, check that you have not accidentally included unnecessary folders (like `node_modules`). If Antigravity generated a project with many extra files, ask your instructor before submitting.

---

## Questions?

Ask before the deadline, not after. Reach out via STiX announcements or during class hours.
